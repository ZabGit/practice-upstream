# practice-upstream

## 📌  Описание проекта

Этот репозиторий создан для отработки навыков работы с Git и GitHub:

- Работа с несколькими удалёнными репозиториями (remote)
- Форк (fork) репозитория
- Синхронизация форка с оригиналом (upstream)
- Создание веток (branches)
- Pull Request и код-ревью
